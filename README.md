 Color Palette Picker

A simple, single-purpose web app that lets users select colors and instantly generate a clean, matching color palette.

The app helps designers, students, and developers quickly find aesthetic color combinations.

 Features

Pick any color using a color picker

Auto-generate matching colors (shades + complementary colors)

Copy color codes with one click

Clean & minimal UI

Works fully offline

Built for Week-1 Micro-Tool challenge
